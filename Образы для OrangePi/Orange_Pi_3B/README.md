* Orangepi3b_1.0.6_ubuntu_jammy_server_linux5.10.160
* Klipper v0.12.0-348
* Moonraker
* Moonraker-timelapse
* Mainsail в качестве вэб интерфейса
* Crowsnest
* KlipperScreen
* Katapult
* Klippain-Shaketune
* Настроен CAN
* Скрипты для автоматического расширения образа на всю MicroSD
